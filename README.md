# White Dove - 白鸽

![White Dove](https://via.placeholder.com/800x400?text=White+Dove+Hero)

**A calm and anonymous emotional support platform for students.**

---

## 🌟 项目介绍

**White Dove** 是一个**匿名情绪支持网站**。  
它提供了一个安全、温柔、无压力的空间，让学生们可以：

- 匿名分享自己的心情和压力
- 写私人心情日记
- 学习简单的放松技巧
- 在焦虑、孤独、 burnout 时找到慰藉

---

## ✨ 主要功能

- 🌐 **双语支持** - English & 中文
- 🕊️ **完全匿名分享** - 无需注册，无痕迹
- 📓 **心情日记** - 私人情绪记录
- 🧘 **放松资源** - 呼吸练习 & Grounding 技巧
- 📱 **完美移动端适配**
- 🎨 **温柔治愈设计** - Apple 风格极简设计

---

## 🛠️ 技术栈

- **HTML5**
- **CSS3** (纯原生，Apple 风格)
- **JavaScript** (Vanilla JS)
- **完全静态网站** - 支持 GitHub Pages
- **本地存储** (LocalStorage)

---

## 🎯 项目目标

这是一个为**大学申请**特别制作的作品，旨在展示：

- 前端开发能力
- 用户体验设计 (UI/UX)
- 对心理健康议题的关注与同理心
- 干净、优雅的代码风格
